# DCN-SDK介绍

## SDK下载地址：
**golang**: <https://github.com/huarong8/dcn_sdk>

## 简介
DCN-SDK实现了区块服务、交易服务等接口，同时还提供了接口使用示例说明，开发者可以调用该SDK方便快捷的生成DCN主链的快速接入
